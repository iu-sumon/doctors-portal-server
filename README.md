 # Doctors-Portal

This is my project live link [Doctors-Portal](https://doctors-portal-2d286.web.app/).

## OverView
 * A complete appointment management portal with a google login feature.
 * There is a dashboard to manage the admin panel and services where an admin can add or remove a doctor and able to manage the appointments.
